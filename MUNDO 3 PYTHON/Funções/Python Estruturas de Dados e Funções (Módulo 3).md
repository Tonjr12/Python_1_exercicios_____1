# 🐍 Python: Estruturas de Dados e Funções (Módulo 3)

Repositório dedicado ao estudo e prática de estruturas avançadas de dados e modularização em Python.

## 🛠️ Conteúdo Desenvolvido
Este conjunto de exercícios cobre a transição entre manipulação de dados em memória e a criação de funções reutilizáveis:

### 1. Manipulação de Dados (Listas e Dicionários)
- **Exercícios 090-092**: Uso de dicionários com validação e ordenação de dados.
- **Exercícios 093-094**: Estruturas compostas (Dicionários dentro de Listas). Implementação de sistemas de cadastro com cópias de memória (`.copy()`) e limpeza de formulários (`.clear()`).

### 2. Funções (Modularização - Aula 20)
- **Conceitos**: Uso de `def` para criação de rotinas, parâmetros, empacotamento (`*args`) e mutabilidade.
- **Área e Contadores**: Criação de funções de cálculo e contadores personalizados com tratamento de erros (`try/except` e sanitização de dados).
- **Lógica Profissional**: Implementação de código independente de variáveis globais, tornando as funções reutilizáveis em outros projetos.

## 🧠 Aprendizado Chave
- **Robustez**: Adição de camadas de segurança (`if p == 0: p = 1`) para evitar erros em tempo de execução.
- **Código Limpo**: Foco na legibilidade e na separação da lógica de processamento (funções) da lógica de exibição (`print`).
- **Escalabilidade**: Preparação do código para ser integrado ao meu sistema de finanças pessoal.
